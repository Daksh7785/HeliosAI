## Terms and Definitions

- **Airflow:** A platform to programmatically author, schedule, and monitor workflows.
- **Captum:** Model interpretability and understanding library for PyTorch.
- **Celery:** An asynchronous task queue/job queue based on distributed message passing.
- **CUSUM (Cumulative Sum):** A sequential analysis technique used for monitoring change detection.
- **FastAPI:** A modern, fast (high-performance), web framework for building APIs with Python 3.7+ based on standard Python type hints.
- **Feast:** An open-source feature store for machine learning.
- **GOES (Geostationary Operational Environmental Satellites):** A network of satellites operated by NOAA, historically the primary source for solar flare classification (A, B, C, M, X).
- **HEL1OS:** High Energy L1 Orbiting X-ray Spectrometer (Payload on Aditya-L1). Measures hard X-rays.
- **Locust:** An easy to use, scriptable and scalable performance testing tool.
- **MLflow:** An open source platform for the machine learning lifecycle.
- **Nowcasting:** The prediction of the present, the very near future, and the very recent past in meteorology.
- **PRADAN:** The ground station data repository for Aditya-L1.
- **SHAP (SHapley Additive exPlanations):** A game theoretic approach to explain the output of any machine learning model.
- **SoLEXS:** Solar Low Energy X-ray Spectrometer (Payload on Aditya-L1). Measures soft X-rays.
- **TimescaleDB:** An open-source time-series database optimized for fast ingest and complex queries, built on PostgreSQL.
